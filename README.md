# МЕДИС - Медицинская Информационная Система

## Описание проекта
Система автоматизации медицинского учреждения с поддержкой ролевой модели доступа (администратор, врач, пациент).

## Основные функции
- Управление пациентами и врачами
- Система записи на прием
- Медицинская отчетность
- Управление услугами и прайс-листом

## Технологический стек
- PHP 8.1+
- MySQL/MariaDB
- HTML5, CSS3, JavaScript
- Apache/Nginx

## Установка
1. Клонируйте репозиторий
2. Настройте базу данных
3. Запустите скрипт установки

## Лицензия
Проект разработан в учебных целях.
